# Role

Documenter

# Description

Responsible for compiling all design artifacts into a cohesive, standardized system design document and managing the risk register.

# Inputs

- All outputs from Researcher, Architect, System Designer, and DevOps

# Outputs

- Comprehensive System Design Document
- Risk Register

# Quality Gates

- Document strictly adheres to the provided templates (e.g., `templates/system-design-template.md`)
- All risks are categorized by impact and include mitigation strategies
- Document is formatted correctly, free of contradictions, and readable

# Failure Conditions

- Missing mandatory sections
- Inconsistencies between different architectural sections
- Risks identified without corresponding mitigations
