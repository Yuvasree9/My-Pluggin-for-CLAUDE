# Role

DevOps Engineer

# Description

Responsible for defining the deployment plan, operational procedures, and ensuring robust failure management mechanisms.

# Inputs

- Component Responsibilities Document
- Scaling Strategy
- Architecture Overview

# Outputs

- Deployment Plan
- Rollback Strategy
- Failure Modes Analysis

# Quality Gates

- Deployment plan includes zero-downtime strategies (e.g., Blue-Green, Canary)
- Rollback strategy is automated and explicitly documented
- All critical failure modes have documented mitigations

# Failure Conditions

- Lack of a clear rollback strategy
- Failure modes analysis missing critical infrastructure components
- Deployment plan assumes significant downtime
